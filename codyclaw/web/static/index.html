Complete the setup to get started

1

飞书应用配置

请先完成以下准备工作：

  1. 打开 飞书开放平台，创建「企业自建应用」
  2. 进入应用 → 「凭证与基础信息」页面，复制 App ID 和 App Secret
  3. 进入「添加应用能力」→ 开启「机器人」，连接方式选择 WebSocket
  4. 进入「权限管理」→ 添加 im:message 权限（收发消息）
在「凭证与基础信息」页面顶部，格式为 cli_ 开头
同一页面中，点击「显示」后复制
在群聊中 @机器人 时用于识别，不填也能正常工作（群聊需 @触发时建议填写）
2

AI 模型配置

配置 AI 模型的 API 密钥。支持 Anthropic（Claude）或其他兼容 OpenAI 接口的服务商。

从 Anthropic Console 获取，或使用其他服务商提供的密钥
仅当使用第三方模型时填写，例如 DeepSeek: https://api.deepseek.com/v1
3

AI 助手设置

给你的 AI 助手起个名字，并指定它的工作目录（Agent 可以在此目录下读写文件和执行命令）。

Agent 执行命令和读写文件的根目录，建议设为你的项目路径
